# Web Calculator

一个简洁美观的网页计算器，支持键盘操作。

## 在线体验

👉 [点击使用](https://suguoming001.github.io/web-calculator)

## 功能

- ✅ 基础四则运算（加、减、乘、除）
- ✅ 支持小数运算
- ✅ 键盘快捷键支持
- ✅ 响应式设计，支持移动端
- ✅ 防止除以零错误
- ✅ 高精度计算

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `0-9` | 输入数字 |
| `.` | 小数点 |
| `+ - * /` | 运算符 |
| `Enter` 或 `=` | 计算结果 |
| `Escape` | 清空 (AC) |
| `Backspace` | 删除 (DEL) |

## 技术栈

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (ES6+)

## 本地运行

```bash
git clone https://github.com/suguoming001/web-calculator.git
cd web-calculator
# 直接用浏览器打开 index.html 即可
```
